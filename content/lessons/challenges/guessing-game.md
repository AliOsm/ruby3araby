# تحدي لعبة التخمين

## مقدمة

في هذا التحدي، ستبني **لعبة تخمين الأرقام**! الكمبيوتر سيختار رقماً عشوائياً، وعلى اللاعب تخمينه. بعد كل محاولة، ستخبر اللاعب إذا كان تخمينه أعلى أو أقل من الرقم الصحيح. هذا التحدي يختبر مهاراتك في الحلقات التكرارية، الجمل الشرطية، والتعامل مع إدخال المستخدم.

## قواعد اللعبة

1. **الكمبيوتر يختار رقماً عشوائياً** بين 1 و 100
2. **اللاعب يُدخل تخميناً**
3. **البرنامج يُعطي تلميحاً:**
   - "أعلى!" إذا كان الرقم الصحيح أكبر من التخمين
   - "أقل!" إذا كان الرقم الصحيح أصغر من التخمين
   - "مبروك!" إذا كان التخمين صحيحاً
4. **تستمر اللعبة** حتى يخمّن اللاعب الرقم الصحيح

## مثال على الناتج المتوقع

```
مرحباً بك في لعبة التخمين!
خمّن رقماً بين 1 و 100
تخمينك: 50
أعلى!
تخمينك: 75
أقل!
تخمينك: 63
أعلى!
تخمينك: 70
مبروك! لقد خمّنت الرقم 70 في 4 محاولات!
```

## المفاهيم المستخدمة

### توليد رقم عشوائي

في Ruby، نستخدم `rand` لتوليد أرقام عشوائية:

```ruby
# رقم عشوائي بين 0 و 99
rand(100)

# رقم عشوائي بين 1 و 100
rand(1..100)

# أو
rand(100) + 1
```

### حلقة التخمين

نستخدم `loop` مع `break` للخروج عند التخمين الصحيح:

```ruby
secret = rand(1..100)

loop do
  puts "تخمينك:"
  guess = gets.chomp.to_i

  if guess == secret
    puts "مبروك!"
    break
  elsif guess < secret
    puts "أعلى!"
  else
    puts "أقل!"
  end
end
```

### عداد المحاولات

```ruby
attempts = 0
secret = rand(1..100)

loop do
  attempts += 1
  guess = gets.chomp.to_i

  if guess == secret
    puts "مبروك! خمّنت الرقم في #{attempts} محاولات!"
    break
  end
  # ...
end
```

### بديل: استخدام while

```ruby
secret = rand(1..100)
guess = nil
attempts = 0

while guess != secret
  puts "تخمينك:"
  guess = gets.chomp.to_i
  attempts += 1

  if guess < secret
    puts "أعلى!"
  elsif guess > secret
    puts "أقل!"
  end
end

puts "مبروك!"
```

## نصائح

1. **ابدأ بتوليد الرقم العشوائي** في بداية البرنامج
2. **استخدم عداداً** لتتبع عدد المحاولات
3. **استخدم `loop do...end` مع `break`** أو `while` للتكرار
4. **قارن التخمين بالرقم السري** لتحديد التلميح المناسب
5. **اطبع عدد المحاولات** عند الفوز

## التعامل مع الإدخال في المتصفح

**ملاحظة مهمة:** في هذا التمرين، نستخدم نظام الإدخال المُحاكى. المدخلات (التخمينات) مُعدّة مسبقاً في حقل الإدخال.

لهذا التمرين:
- الرقم السري **ثابت** (وليس عشوائياً) لنتمكن من التحقق من الناتج
- التخمينات هي: `50`, `75`, `60`, `65`
- الرقم السري هو: `65`

## التمرين

حان دورك! اكتب لعبة تخمين الأرقام:

1. استخدم رقماً سرياً ثابتاً (`secret = 65`) للتحقق
2. استقبل التخمينات من المستخدم
3. أعطِ تلميحاً بعد كل محاولة: "أعلى!" أو "أقل!"
4. عند التخمين الصحيح، اطبع رسالة فوز مع عدد المحاولات

**تذكير:** في التطبيق الحقيقي، ستستخدم `rand(1..100)` بدلاً من رقم ثابت!
