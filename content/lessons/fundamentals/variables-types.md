# المتغيرات وأنواع البيانات

في البرمجة، نحتاج إلى تخزين المعلومات لاستخدامها لاحقاً. هنا يأتي دور **المتغيرات** (Variables)! المتغير هو مثل صندوق تضع فيه قيمة ما وتعطيه اسماً لتتمكن من الرجوع إليه.

## إنشاء المتغيرات

في روبي، إنشاء المتغيرات بسيط جداًّ. لا تحتاج لتحديد نوع البيانات - روبي تكتشفه تلقائياً.

```ruby
name = "أحمد"
age = 25
puts name
puts age
```

الناتج:
```
أحمد
25
```

لاحظ أننا استخدمنا علامة `=` لتعيين القيمة للمتغير. هذا يسمى **عامل التعيين** (Assignment Operator).

## قواعد تسمية المتغيرات

لتسمية المتغيرات في روبي، اتبع هذه القواعد:

1. **تبدأ بحرف صغير أو شرطة سفلية** `_`
2. **تحتوي على حروف وأرقام وشرطات سفلية فقط**
3. **لا تحتوي على مسافات**
4. **لا تبدأ برقم**

```ruby
# أسماء صحيحة
user_name = "سارة"
age2 = 30
_private = "خاص"

# أسماء خاطئة (ستسبب خطأ)
# لا تبدأ برقم
# 2age = 30
# لا مسافات
# user name = "x"
# لا شرطات عادية
# user-name = "x"
```

### اصطلاح snake_case

في روبي، نستخدم **snake_case** لتسمية المتغيرات - أي نفصل الكلمات بشرطة سفلية:

```ruby
first_name = "محمد"
last_name = "علي"
full_name = "محمد علي"
user_age = 28
```

## أنواع البيانات الأساسية

روبي تدعم عدة أنواع من البيانات. دعنا نتعرف على أهمها:

### 1. النصوص (Strings)

النص هو سلسلة من الحروف محاطة بعلامات اقتباس:

```ruby
greeting = "مرحبا"
name = 'أحمد'
message = "مرحبا يا صديقي!"

puts greeting
puts name
puts message
```

الناتج:
```
مرحباً
أحمد
مرحباً يا صديقي!
```

يمكنك استخدام علامات اقتباس مزدوجة `"` أو مفردة `'`. الفرق الرئيسي هو أن علامات الاقتباس المزدوجة تدعم **الدمج** (interpolation) والرموز الخاصة مثل `\n` للسطر الجديد.

### 2. الأرقام الصحيحة (Integers)

الأرقام الصحيحة هي أرقام بدون فاصلة عشرية:

```ruby
age = 25
year = 2025
negative_number = -10
# يمكن استخدام الشرطة السفلية _ لتسهيل قراءة الأرقام الكبيرة
big_number = 1_000_000

puts age
puts year
puts negative_number
puts big_number
```

الناتج:
```
25
2025
-10
1000000
```

لاحظ أن `1_000_000` تساوي مليون - الشرطة السفلية للقراءة فقط ولا تؤثر على القيمة.

### 3. الأرقام العشرية (Floats)

الأرقام العشرية تحتوي على فاصلة عشرية:

```ruby
price = 19.99
pi = 3.14159
temperature = -5.5

puts price
puts pi
puts temperature
```

الناتج:
```
19.99
3.14159
-5.5
```

### 4. القيم المنطقية (Booleans)

القيم المنطقية تمثل الصواب والخطأ - إما `true` أو `false`:

```ruby
is_student = true
is_working = false

puts is_student
puts is_working
```

الناتج:
```
true
false
```

هذه القيم مهمة جداًّ في الشروط والقرارات البرمجية.

### 5. القيمة الفارغة (nil)

`nil` تعني "لا شيء" أو "فارغ". تُستخدم عندما لا توجد قيمة:

```ruby
unknown = nil

puts unknown.inspect
puts unknown.nil?
```

الناتج:
```
nil
true
```

`nil.nil?` تُرجع `true` لأن القيمة فارغة!

## معرفة نوع البيانات

يمكنك معرفة نوع أي قيمة باستخدام `.class`:

```ruby
puts "مرحباً".class
puts 42.class
puts 3.14.class
puts true.class
puts nil.class
```

الناتج:
```
String
Integer
Float
TrueClass
NilClass
```

## تغيير قيم المتغيرات

المتغيرات قابلة للتغيير - يمكنك تحديث قيمتها في أي وقت:

```ruby
score = 0
puts score

score = 10
puts score

score = score + 5
puts score
```

الناتج:
```
0
10
15
```

### اختصارات التحديث

روبي توفر اختصارات لعمليات التحديث الشائعة:

```ruby
count = 10

# يساوي: count = count + 5
count += 5
# 15
puts count

# يساوي: count = count - 3
count -= 3
# 12
puts count

# يساوي: count = count * 2
count *= 2
# 24
puts count
```

الناتج:
```
15
12
24
```

## أمثلة عملية

### مثال 1: معلومات شخصية

```ruby
name = "فاطمة"
age = 22
height = 1.65
is_student = true

puts "الاسم: " + name
puts "العمر: " + age.to_s
puts "الطول: " + height.to_s
puts "طالبة: " + is_student.to_s
```

الناتج:
```
الاسم: فاطمة
العمر: 22
الطول: 1.65
طالبة: true
```

لاحظ استخدام `.to_s` لتحويل الأرقام والقيم المنطقية إلى نص للدمج!

### مثال 2: حساب بسيط

```ruby
product_name = "كتاب"
price = 50
quantity = 3
total = price * quantity

puts "المنتج: " + product_name
puts "السعر: " + price.to_s
puts "الكمية: " + quantity.to_s
puts "الإجمالي: " + total.to_s
```

الناتج:
```
المنتج: كتاب
السعر: 50
الكمية: 3
الإجمالي: 150
```

## جدول أنواع البيانات

| النوع | الاسم بالإنجليزية | أمثلة | الاستخدام |
|-------|------------------|-------|-----------|
| نص | String | `"مرحبا"`, `'hello'` | تخزين النصوص |
| رقم صحيح | Integer | `42`, `-10`, `1_000` | الأعداد الصحيحة |
| رقم عشري | Float | `3.14`, `-0.5` | الأرقام العشرية |
| منطقي | Boolean | `true`, `false` | القرارات والشروط |
| فارغ | NilClass | `nil` | غياب القيمة |

## نصائح مهمة

1. **اختر أسماء واضحة للمتغيرات** - `user_age` أفضل من `x`
2. **استخدم snake_case** - `first_name` وليس `firstName`
3. **روبي تكتشف النوع تلقائياً** - لا تحتاج لتحديده
4. **المتغيرات قابلة للتغيير** - يمكنك تحديث قيمتها

## تمرين: أنشئ متغيرات من أنواع مختلفة

حان وقت التطبيق! في محرر الشيفرة على اليسار:

**المطلوب:**
1. أنشئ متغير نصي باسم `city` يحتوي على "القاهرة"
2. أنشئ متغير رقمي صحيح باسم `population` يحتوي على 20000000
3. أنشئ متغير عشري باسم `area` يحتوي على 3085.1
4. أنشئ متغير منطقي باسم `is_capital` يحتوي على true
5. اطبع كل متغير باستخدام puts

الناتج المتوقع:
```
القاهرة
20000000
3085.1
true
```

> **تلميح:** تذكر أن روبي تكتشف نوع البيانات تلقائياًّ - فقط اكتب القيم مباشرة!

---

> **تذكّر:** المتغيرات هي أساس كل برنامج - بدونها لا يمكننا تخزين أو معالجة أي بيانات!
