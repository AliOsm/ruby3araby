# التعليقات في روبي

التعليقات هي نصوص في الشيفرة لا يُنفذها المُفسِّر (Interpreter). نستخدمها لشرح الشيفرة وتوثيقها للمبرمجين الآخرين أو لأنفسنا في المستقبل.

## لماذا نستخدم التعليقات؟

- **شرح الشيفرة**: توضيح ما يفعله جزء معين من الشيفرة
- **التوثيق**: تسجيل معلومات عن البرنامج ومُنشئه
- **تعطيل الشيفرة مؤقتاً**: إيقاف تنفيذ أسطر معينة أثناء الاختبار
- **التخطيط**: كتابة الخطوات قبل كتابة الشيفرة الفعلية

## التعليق أحادي السطر

لكتابة تعليق في سطر واحد، نستخدم علامة `#` (الشباك). كل ما يأتي بعد هذه العلامة في نفس السطر يُعتبر تعليقاً:

```ruby
# هذا تعليق كامل في سطر واحد
# هذا تعليق في نهاية السطر
puts "مرحبا"
```

الناتج:
```
مرحبا
```

لاحظ أن التعليقات لا تظهر في الناتج!

## أمثلة على التعليقات أحادية السطر

```ruby
# برنامج الترحيب
# الإصدار 1.0

# طباعة رسالة ترحيب
puts "أهلاً وسهلاً!"

# يمكنك وضع التعليق بعد الشيفرة
# تخزين اسم المستخدم
name = "أحمد"
puts name
```

## التعليق متعدد الأسطر

عندما تحتاج لكتابة تعليق طويل يمتد على عدة أسطر، يمكنك استخدام `=begin` و `=end`:

```ruby
=begin
هذا تعليق متعدد الأسطر.
يمكن أن يحتوي على عدة فقرات
لشرح جزء معقد من الشيفرة.
اسم البرنامج: الآلة الحاسبة
المؤلف: فريق التطوير
=end

puts "البرنامج يعمل!"
```

### قواعد مهمة للتعليقات متعددة الأسطر

1. يجب أن تكون `=begin` و `=end` في بداية السطر تماماً (لا مسافات قبلها)
2. كل ما بينهما يُعتبر تعليقاً ولا يُنفَّذ

```ruby
=begin
كل هذا النص
هو تعليق
ولن يُنفَّذ
=end
puts "تم!"
```

## أيهما أستخدم؟

| الحالة | النوع المُفضَّل |
|--------|----------------|
| تعليق قصير | `#` |
| شرح سطر واحد | `#` |
| توثيق طويل | `=begin...=end` |
| تعطيل شيفرة مؤقتاً | `#` لأسطر قليلة |

في الواقع العملي، معظم مبرمجي روبي يستخدمون `#` حتى للتعليقات الطويلة:

```ruby
# هذه الشيفرة تحسب مجموع الأرقام
# من 1 إلى 100 باستخدام معادلة غاوس
# المعادلة: n * (n + 1) / 2
n = 100
result = n * (n + 1) / 2
puts result
```

## نصائح لكتابة تعليقات جيدة

1. **اكتب لماذا وليس ماذا**: الشيفرة تُظهر "ماذا" يحدث، التعليق يشرح "لماذا"
2. **اجعلها محدثة**: تعليق قديم أسوأ من عدم وجود تعليق
3. **اختصر**: التعليق الجيد واضح ومختصر
4. **استخدم العربية أو الإنجليزية باستمرار**: اختر لغة واحدة للتعليقات

## تعطيل الشيفرة مؤقتاً

أحياناً نريد إيقاف تنفيذ سطر معين لاختبار شيء ما:

```ruby
puts "السطر الأول"
# puts "هذا السطر معطّل"
puts "السطر الثالث"
```

الناتج:
```
السطر الأول
السطر الثالث
```

## تمرين: استخدام التعليقات

حان وقت التطبيق! في محرر الشيفرة على اليسار:

**المطلوب:** لديك شيفرة تطبع ثلاثة أسطر. أضف تعليقاً قبل السطر الثاني لتعطيله، بحيث يُطبع السطران الأول والثالث فقط.

الناتج المتوقع:
```
روبي
رائعة
```

> **تلميح:** ضع علامة `#` في بداية السطر الذي تريد تعطيله.

---

> **تذكّر:** التعليقات الجيدة تجعل الشيفرة أسهل للفهم والصيانة. اعتد على كتابتها منذ البداية!
